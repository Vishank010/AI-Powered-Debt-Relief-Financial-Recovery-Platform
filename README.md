# AI-Powered-Debt-Relief-Financial-Recovery-Platform

FinRelief AI is an AI-powered web application that helps borrowers manage loan 
accounts, analyze their financial health, receive AI-driven settlement 
predictions, and generate lender-specific negotiation letters. Built with 
React.js, FastAPI, SQLite, SQLAlchemy, and Google Gemini API, it features JWT 
authentication, real-time financial health scoring (EMI/DTI ratios, debt stress 
level), rule-based settlement prediction, and AI-generated negotiation strategies 
with automatic fallback logic when the AI service is unavailable.
