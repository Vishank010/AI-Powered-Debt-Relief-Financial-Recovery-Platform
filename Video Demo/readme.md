# Video Demo – FinRelief AI

This folder contains a screen-recorded demo of the **FinRelief AI – Debt Relief 
& Financial Recovery Platform**, showing the complete working application.

## File
- `14.07.2026_18.42.39_REC.mp4`

## What the Demo Covers
1. User registration and login (JWT-based authentication)
2. Adding and managing loan accounts
3. Financial Health Dashboard – EMI ratio, DTI ratio, monthly surplus, and 
   debt stress level
4. AI-powered Settlement Prediction for a loan account
5. AI Negotiation Letter Generator – strategy and settlement letter generation
6. AI History log showing past predictions and generated letters

## Tech Stack Demonstrated
React.js (Vite) frontend + FastAPI backend + SQLite database + Google Gemini 
API integration (with automatic rule-based fallback).
